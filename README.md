# ☕ Coffee Store Sales Dashboard — Excel

## 📌 Overview
An end-to-end Excel project analyzing coffee shop sales data — from raw data cleaning to a fully interactive dashboard. This project uncovers sales trends, peak hours, top-selling products, and store performance to support data-driven business decisions.

---

## 🛠️ Tools & Technologies
![Excel](https://img.shields.io/badge/Microsoft_Excel-217346?style=for-the-badge&logo=microsoft-excel&logoColor=white)

---

## 🎯 Objectives
- Clean and transform raw sales data into analysis-ready format
- Identify peak sales hours and busiest days of the week
- Analyze revenue by product category and store location
- Build an interactive dashboard with slicers and pivot charts

## 📊 Dashboard Features

| Visual | Description |
|---|---|
| 📈 Revenue Over Time | Monthly revenue trend line chart |
| ⏰ Sales by Hour | Peak hour analysis bar chart |
| 📅 Sales by Day of Week | Busiest days comparison |
| 🏪 Sales by Store Location | Revenue breakdown per store |
| ☕ Sales by Product Category | Top-performing product categories |
| 🔢 KPI Summary | Total Revenue, Total Orders, Avg Order Value |
